# 程式設計(二) 期末專題
## 何茹瑄 409570080

## 專題用途
可以透過玩樂的方式來知道一些登山小知識
## 使用方式
在Hiking開始程式後，會跳出MainMenu的視窗，裡面有四個按鈕，
第一個是小遊戲，可以用玩樂的方式來知道一些小知識，遊戲結束後
可以知道自己的答對跟答錯數，第二是題庫，可以觀看整個題庫的題目，
第三個是參考資料的連結，只要點文字就能連到相關網站，最後是結束
整個程式
